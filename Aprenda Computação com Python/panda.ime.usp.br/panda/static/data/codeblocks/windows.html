Instalação do Code::Blocks no Windows

Code::Blocks é um ambiente integrado de desenvolvimento para as linguagens C e C++. Ele usa a implementação Mingw do GCC (GNU Compiler Collection) como seu compilador. Ele é "Free Software" (sob GNU General Public License). Isso significa, entre outras coisas, que ele pode ser distribuído e copiado à vontade.

Você pode utilizar outro compilador se desejar, mas deve se certificar que os seus programas podem ser compilados pela versão do Code::Blocks disponível nessa página e na sala Pró-Aluno antes de entregá-los.

As telas abaixo foram geradas no Windows 7 em inglês. A instalação pode ser ligeiramente diferente para outras versões do Windows e se o idioma do seu Windows for outro.

Depois de instalar o Code::Blocks veja a Aula Dirigida de Uso do Code::Blocks.

 

Instalação Básica do Code::Blocks

  1. Baixe o arquivo de instalação daqui codeblocks-16.01mingw-setup.exe [ 79.9MB ou 83783938 bytes ] para uma pasta temporária. Esse arquivo pode ser removido depois da instalação. Depois de clicar no link acima será pedida uma confirmação de que você realmente quer baixar o arquivo. Clique em Save File para continuar. A figura abaixo é da versão 13.12, mas não se preocupe com isso.

  2. Depois de baixado o instalador, ele deve ser executado. Para tanto, clique com o botão direito do mouse na janela de Downloads e escolha Open Containing Folder.

  3. Em seguida, para começar a instalação basta clicar duas vezes (double click) no instalador codeblocks-16.01mingw-setup.exe. Se for pedida uma confirmação para a execução do instalador, clique em Run para continuar.

  4. Durante a instalação escolha sempre as opções sugeridas pelo instalador. Clique em Next para o Wizard.

  5. Na janela da lincença clique em I agree.

  6. Clique em Next para o Choose Components.

  7. Clique em Install para o Choose Install Location.

Configurando o Code::Blocks

  1. Ao final da instalação, aparecerá a tela abaixo. Você deverá clicar em Yes para executar o Code::Blocks.

  2. A seguir, certifique se de que na próxima tela, Compilers auto-detection, o compilador selecionado seja o GCC. Na tela baixo veja a mensagem "Current default compiler: GNU GCC compiler". Depois clique em OK.

  3. Na janela que aparecer, clique em OK para a escolha Yes, associate Code::Blocks with C/C++ file types, como indicado abaixo.

  4. Agora, selecione Settings... e depois Compiler, como indicado abaixo.

  5. Na tela que aparecer, assinale as 4 opções indicadas abaixo (-ansi, -Wall, -pedantic e -O2) e clique em OK.

  6. Para finalizar, primeiro feche a janela do Code::Blocks. Na tela que aparecer, clique em Yes.

  7. Observe que ainda sobrou uma janela do processo de instalação. Clique em Next.

  8. Finalmente, clique em Finish para encerrar o programa.

  9. Divirtam-se!
