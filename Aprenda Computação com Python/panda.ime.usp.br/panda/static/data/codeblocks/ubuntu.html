Instalação do Code::Blocks no Ubuntu Linux

Code::Blocks é um ambiente integrado de desenvolvimento para as linguagens C e C++. Ele é um "Free Software" (sob GNU General Public License). Isso significa, entre outras coisas, que ele pode ser distribuído e copiado à vontade.

Durante o curso você pode utilizar outro compilador se desejar, mas deve se certificar que os seus programas podem ser compilados pela versão do Code::Blocks disponível nessa página e na sala Pró-Aluno antes de entregá-los.

Depois de instalar o Code::Blocks veja a Aula Dirigida de Uso do Code::Blocks.

 

Instalação do Code::Blocks no Ubuntu 14.04

Área de trabalho do Ubuntu do usuário hofstadter .

Abra uma janela de comandos, também conhecida como terminal ou console. No Ubuntu, tecle 

Ctrl+Alt t
Abertura de um terminal na área de trabalho.

No terminal (janela que abriu), digite o seguinte comando:

sudo apt-get install codeblocks
e tecle ENTER

Comando para instalar o Code::Blocks.

Esse comando vai perguntar por sua senha (você precisa ter permissão de administrador para poder instalar programas).

Pedido da senha do usuário.
Digite sua senha e, caso você tenha permissão, o Ubuntu vai automaticamente buscar o programa e instalá-lo em seu computador e... PRONTO!

Para iniciar o Code::Blocks caso tenha sucesso na instalação, em um terminal, digite 

codeblocks &
Comando para executar o Code::Blocks.

Depois disto o Code::Blocks se abrirá.

Janela inicial Code::Blocks.

Na primeira vez que você utilizar o Code::Blocks, o programa vai perguntar se você deseja utilizar o compilador default (no caso o GCC). Aceite essa opção clicando em Ok.

Detecção automática do GCC.

Além da tela principal, o Code::Blocks oferece uma janela com "dicas" para melhor utilizar a ferramenta. Você pode fechá;-la, ou minimizá-la, ou até mesmo explorar as dicas se desejar.

Dicas do Code::Blocks.

Agora vamos configurar o Code::Blocks. Na janela do Code::Blocks vá com o cursor até o topo da tela. Uma barra com opções surgirá. Nessa barra siga 

Settings --> Compiler
Janela com opções de compilação do Code::Blocks.

A seguir, na tela que aparecer, assinale as 4 opções indicadas abaixo e clique em OK.

Opções de compilação -ansi, -Wall e -pedantic.

 

Opção de compilação -O2.

Finalmente, clique na aba "Other options" e na janela que é mostrada escreva -Wno-unused-result e depois clique em OK.

Opção de compilação -Wno-unused-result.

 

Agora é só correr para o abraço e se divertir! Veja agora a Aula Dirigida de Uso do Code::Blocks.
