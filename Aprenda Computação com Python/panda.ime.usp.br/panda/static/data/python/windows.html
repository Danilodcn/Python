Instalação do Python no Windows

A implementação da linguagem Python é licenciada como open source license. Isto significa, entre outras coisas, que ela pode ser distribuída e copiada à vontade.

Existem implementações de Python para vários tipos de sistemas operacionais (linux, Mac OS, Windows,...). Aqui está descrita uma instalação para Microsoft Windows. De acordo com as suas necessidades, você pode também querer obter uma versão portátil que roda o Python de um pendrive: http://portablepython.com/.

Na página da distribuição das implementações, existem hoje duas versões do Python: Python 3.x.y e Python 2.w.z. A versão que utilizaremos é a Python 3.x.y. O Python utilizado é o 3, os número x e y são irrelevantes, no entando sugerimos que você instale sempre a versão "instável" mais recente do Python disponível. Por exemplo, no momento em que esta página foi criada, a versão mais recente era a Python 3.3.0 e em 19/02/2014 a versão mais recente era a Python 3.3.2. As ilustrações abaixo mostram a instalação do Python 3.3.2, mas no momento em que você estiver instalando o Python é possível que a versão seja outra. De qualquer maneira a instalação será muito semelhante.

Uma alternativa, que também pode ser usada de forma complementar, é usar uma instalação de Python em um pen drive. Aí, você pode facilmente ir desenvolvendo seu programa em vários computadores, sem necessidade de instalar o Python em nenhum deles. A página para baixar esse instalador de Python em pen drive é Portable Python. Sempre escolha para abaixar a versão mais nova do programa, que neste momento é a Portable Python 3.2.5.1.

As telas abaixo foram geradas no Windows 7 em inglês. A instalação pode ser ligeiramente diferente para outras versões do Windows e se o idioma do seu Windows for outro.

Instruções para a Instalação

Em poucas palavras, a instalação consiste em baixar o instalador do Python e selecionar uma dentre as opções sugeridas pelo instalador. A seguir descrevemos a instalação em mais detalhes.

  1. Acesse a página python.org/download/ e baixe o instalador apropriado para a sua versão de windows. É um dos dois seguintes, dependendo da sua versão do windows.

    • Python 3.x.y Windows x86 MSI Installer (Windows binary -- does not include source)
    • Python 3.x.y Windows X86-64 MSI Installer (Windows AMD64 / Intel 64 / X86-64 binary -- does not include source)

    Se você não tem certeza de qual é a sua versão do windows, escolha a primeira opção, o Python 3.x.y Windows x86 MSI Installer.

    Quando está página foi criada em 2013 a página para baixar o Python tinha a aparência um pouco diferente e a versão mais recente era a 3.3.0. As ilustrações abaixo mostram os passos da instalação do Python 3.3.2. Esses passos devem ser semelhantes aos passos de outras versões.

  2. Será pedida uma confirmação de que você realmente quer baixar o arquivo. Clique em Save File para continuar.

  3. Depois de baixado o instalador, ele deve ser executado. Para tanto, clique com o botão direito do mouse na janela de Downloads e escolha Open Containing Folder.

  4. Em seguida, se na sua instalação aparecer uma tela como a que está a eguir click duas vezes (double click) no instalador python-3...

  5. Será pedida uma confirmação para a execução do instalador. Clique em Run para continuar.

  6. A seguir, deverá ser escolhido se a instalação deve ser feita a todos os usuários do computador ou somente para você. No Windows Vista só existe realmente a opção de instalar para todos os usuários. Seja gentil e instale Python para todos ;-). Clique em Next para continuar.

  7. Na tela a seguir, escolha as opções sugeridas clicando em Next. Na dúvida, escolha a opção sugerida que deverá ser satisfatória na maioria dos casos.

  8. Novamente, escolha as opções sugeridas ;-) clicando em Next.

  9. A seguir, antes de começar a instalação, será pedido para você autorizar que o programa seja executado. Você verá uma tela com o título "Clique em Continue para seguir para o próximo passo.

  10. Uma tela mostrando o progresso da instalação será mostrada. Após algum tempo, a tela abaixo indica a conclusão da instalação. Clique em Finish para concluir a instalação.

  11. É possível que o Windows peça permissão para reiniciar o seu computador para que as alteração nas configuração devidas a instalação sejam efetivadas. Clique em Yes.

  12. Para testar a instalação, depois de uma possível reinicialização do Windons, abra o programa IDLE, conforme indicado a seguir.

  13. O programa IDLE deve abrir. Brinque um pouco como o interpretador digitando algumas contas.

  14. Divirtam-se!
