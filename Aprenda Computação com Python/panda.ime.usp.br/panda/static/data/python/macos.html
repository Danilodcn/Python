Instalação do Python no MacOS

A implementação da linguagem Python é licenciada como open source license. Isto significa, entre outras coisas, que ela pode ser distribuída e copiada à vontade.

Existem implementações de Python para vários tipos de computadores. Aqui está descrita uma instalação para o MacOS.

Na página da distribuição das implementações, existem hoje duas versões do Python: Python 3.x.y e Python 2.w.z. Você deve sempre usar a última versão disponível do Python. Por exemplo, no momento em que esta página foi escrita, a versão mais recente era a Python 3.3.4.

Além do python, é necessário instalar algum pacote Tcl/Tk para que você consiga utilizar a interface gráfica para interagir com o python. A versão a ser instalada depende da versão do seu sistema operacional. Para verificar a versão do MacOS do seu computador, clique na maçã localizada no canto superior esquerdo da barra de ferramentas e, no menu que for exibido, clique em "Sobre este mac":




O vídeo abaixo mostra a instalação do Python 3.3.4 e do ActiveTcl 8.6.1 em um computador rodando o MacOS versão 10.9.1 em português. Pode ser que no momento em que você esteja instalando o Python e o ActiveTCL, as versões mais novas deles já sejam outras. A instalação pode ser ligeiramente diferente para outras versões e idiomas do MacOS.
