Como escrever um programa em Python

A maneira mais simples para escrever um programa em Python é utilizando um editor de texto. Assim, usando qualquer editor, você pode criar um programa e salvá-lo em um arquivo (com extesão .py).

Para executar o programa basta digitar "python nome_do_programa.py" em um terminal, a partir do mesmo diretório onde o programa foi salvo.

Os editores de texto comuns, no entanto, não oferecem muitos recursos para edição de programas em Python. Embora existam vários ambientes de desenvolvimento para Python, vamos mencionar apenas 2, o IDLE e o Spyder. O IDLE é um ambiente integrado de desevolvimento de programas (ou IDE do inglês Integrated Development Environment) em Python relativamente simples de utilizar. O tutorial O ABC do IDLE oferece uma breve introdução ao IDLE e seus recursos.
