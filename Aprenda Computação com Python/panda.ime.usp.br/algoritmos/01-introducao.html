Introdução - Resumo de Python¶

Como alguns alunos tenham, possivelmente, utilizado outra linguagem durante o curso introdutório e também para ajudar aqueles alunos que fizeram o curso introdutório em Python mas que se acham um tanto “enferrujados”, vamos nessa aula cobrir alguns conceitos básicos de Python para nivelar todo mundo e continuar o processo de desenvolvimento do raciocínio aplicado na formulação e resolução de problemas computacionais.

Introdução¶

Os exercícios e programas que vamos desenvolver nessa disciplina serão baseados na versão 3.x da linguagem de programação Python. Siga as instruções de instalação do Python para instalar a versão mais recente do Python 3.x em seu computador.

O material dessas aulas foi desenvolvido utilizando as ferramentas do Projeto Runestone. Essas ferramentas permitem que você desenvolva os seus programas e teste a execução dos mesmos dentro do seu próprio navegador. Para saber conhecer melhor essas ferramentas leia a seção Formas especiais para rodar Python nesse livro do livro Como Pensar como um Cientísta da Computação - Aprendendo com Python: Edição interativa.

Exercício comentado 1 - fatorial¶

Os objetivos com este exercício são apresentar:

  • o esqueleto dos programas nessa disciplina: sempre utilize esse esqueleto em seus programas;
  • as funções de entrada e saída: print() e input(); e
  • comandos básicos de seleção e repetição.

(funcao_fatorial_iterativa)

A tabulação dos comandos em Python é muito importante pois comandos com a mesma tabulação estão dentro do mesmo bloco. Portanto, o corpo da função main() precisa ficar deslocado de um tab. O texto entre apóstrofes triplos é chamado de docstring. Recomendamos o uso de docstrings para documentar o seu programa.

A função input() imprime o texto que recebe como argumento como mensagem ao usuário do programa, e espera o usuário digitar a resposta no teclado. Essa função sempre retorna um string, que é então convertido para um int (número inteiro) – mas poderia também ser convertido para um número real usando a função float().

O comando if testa se o número é maior ou igual zero, e nesse caso calcula o fatorial (usando a função fatorial() definida no próprio programa, mais abaixo) e imprime o resultado usando a função print(). Caso o número seja negativo, o else é executado. Essa sintaxe deve lhe ser familiar e muito parecida com a sintaxe da linguagem que você utilizou no curso introdutório.

Observe que a função fatorial() é definida fora da função main(). A última linha do esqueleto (observe que essa linha também não faz parte do corpo da função fatorial()) chama a função main() para que o programa principal seja executado.

Em seus programas, procure utilizar essa mesma estrutura:
  • defina a função main() primeiro
  • defina as demais funções que você precisa utilizar, sem se preocupar com a ordem de definição dessas funções.
  • chame a função main()

Clique em Run para executar o programa. Essa região em seu navegador que lhe permite ver e executar o programa chama-se activecode. O activecode também lhe permite editar o programa. Tente fazer o programa a seguir no próprio activecode.
